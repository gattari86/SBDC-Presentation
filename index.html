TROUBLESHOOTING

Common Issues + Fixes

Quick solutions to the most common problems

Google Form Issues

Form not saving

Browser or connection issue

✓ Refresh page, changes auto-save

Can't find spreadsheet

Sheet not created yet

✓ Go to Responses tab, click Sheets icon

Responses not showing

Form not accepting responses

✓ Check "Accepting responses" ON; use Settings tab

Make.com Issues

Connection failed

Google permissions expired

✓ Click connection, re-authorize

No data found

No new rows since connecting

✓ Submit a new test form first

Scenario won't run

Out of ops, not active, or set to On demand

✓ Check toggle ON; On demand only runs via Run once

Email Issues

Email not sending

Gmail not connected

✓ Re-add Gmail connection

Going to spam

Generic subject or content

✓ Personalize with recipient's name

Missing data in email

Fields not mapped correctly

✓ Re-map from Google Sheets bubbles
